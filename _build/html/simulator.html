    guided_simulation(source_marking, state_mapping, relation)¶

    Simulate starting from the source_marking until reaching a final marking. Follows the path of init first, then directly the path of successive from the relation parameter.

    Parameters:

    • source_marking – Initial marking

    • state_mapping – “global” marking to keep track of the markings

    • relation – relation to discover with guided simulation

    Returns:

    list of discovered traces

    random_simulation(initial_marking, state_mapping, guided_traces=None, source_marking=None)¶

    Perform a random execution of the model, in this case a petrinet, and returns the list of discovered traces

    Parameters:

    • initial_marking – Initial marking, where to start

    • state_mapping – “global” marking to keep track of the markings

    • guided_traces – If you already have starting traces, default None

    • source_marking – If you already have starting traces you have to include their marking, default None

    Returns:

    list of discovered traces
